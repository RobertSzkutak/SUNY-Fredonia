K-Means Clustering Visualizer

Written by Robert Szkutak

Origin

Usage

kmcv was written as a utility to aid an undergraduate research project at SUNY Freodnia in the Fall 2012 semester. Development has continued throughout the Fall 2013 semester through further undergraduate research. kmcv is open source software and is completely free for anyone to use.

kmcv performs the K-Means clustering algorithm on any given set of two-dimensional data and then renders the output in plaintext, an SDL-based desktop client, and/or HTML5. kmcv supports an extensive suite of options and features for it's functionality to be adapted to any kind of data analysis.
